서버 소개

라인서버는 “몰입감”과 “편의성”을 같이 잡는 방향으로 간다.

우리가 지향하는 RP

  • 몰입을 깨는 요소 최소화
  • 룰 기반 운영 + 빠른 판정
  • 뉴비 친화 가이드/도움 구조
  • 라인 감성 UI로 가독성 극대화

서버 시스템

ESX Legacy OXMYSQL 커스텀 HUD 커스텀 채팅 디스코드 연동(예정) 포털/관리자(예정)

* 포털/관리자는 2차에서 Firebase/DB 연동으로 실제 기능 붙이면 된다.

규칙

핵심만 요약. 자세한 건 디스코드 규칙 채널로 연결하면 완벽.

NonRP / BadRP

현실 감각/상황을 무시하는 행동, RP 흐름을 망치는 행동.

  • 논RP 운전/무리한 드리프트/무지성 박치기
  • 상황 파괴(강제 RP 유도, 반복 트롤)

MG / PG

메타 정보(현실/외부) 사용, 비현실적인 강행동.

  • 디스코드로 상황 공유/현실 정보 사용
  • 불가능한 행동을 “한다”로 밀어붙이기

RDM / VDM

이유 없는 살해/차량 살해 행위.

  • 이유 없는 공격/총기 사용
  • 무차별 납치/무차별 살인

시스템/버그 악용

시스템의 허점을 이용해 이득 취하는 행동.

  • 버그/악용/테러 행위 금지
  • 경찰·의료 RP 방해 금지
상세 규칙은 디스코드에 고정 사이트엔 요약, 디코엔 전문(버전 관리)
